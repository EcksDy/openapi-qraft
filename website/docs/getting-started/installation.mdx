# Installation

Before you start using OpenAPI Qraft, you need to install the Qraft binding for the React
and the CLI tool to generate the API client. You will also need to install
[`openapi-typescript`](https://www.npmjs.com/package/openapi-typescript) to generate the types from the OpenAPI Document.

```bash npm2yarn
npm install @openapi-qraft/react
npm install -D @openapi-qraft/cli
npm install -D openapi-typescript
```

## Requirements

OpenAPI Qraft is compatible with React v18+ and works with ReactDOM and React Native.
Browser support is the sa as requirements as [Tanstack Query](https://tanstack.com/query/latest/docs/framework/react/installation#requirements):

```text
Chrome >= 91
Firefox >= 90
Edge >= 91
Safari >= 15
iOS >= 15
Opera >= 77
```

:::info Peer Dependency

If you are not already using **Tanstack Query v5**, you need to
[install](https://tanstack.com/query/latest/docs/framework/react/installation) it as well:

```bash npm2yarn
npm install @tanstack/react-query
```

:::
