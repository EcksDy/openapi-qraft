# Installation

Before you start using OpenAPI Qraft, you need to install the CLI tool to generate the API client.
You also need to install the Qraft Client React package to use the generated API client.

```bash npm2yarn
npm install -D @openapi-qraft/cli@next
npm install @openapi-qraft/react@next
```

:::info Peer Dependency

If you are not already using **TanStack Query v5**, you need to
[install](https://tanstack.com/query/latest/docs/framework/react/installation) it as well:

```bash npm2yarn
npm install @tanstack/react-query
```

:::
